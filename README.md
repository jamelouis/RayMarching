# RayMarching
  在PC端仿ShaderToy搭了个简易的RayMarching的框架, 便于RayMarching的学习。
  
  目前，已将ShaderToy的SunSurface, Cloud两个很酷的例子集成到框架中。
  
## RayMarching的框架简介
  - ray-marching.ini: 配置当前是哪个实例
  - DemoConfig/*.ini:  实例的具体配置
    + fragment shader path
    + iChannel0 texture path
  - Texture/: 纹理所在的路径
  - Shader/: 着色器所在的路径
